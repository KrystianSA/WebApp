## Getting Started


To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo

```sh
git clone https://github.com/KrystianSA/WebApp.git
```

2. Install NPM packages

```sh
npm install
```

3. Enter your API in `environment.ts`

```sh
baseUrl: ' '
```
